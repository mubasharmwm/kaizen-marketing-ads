## Production Notes for All Creatives

1. **Text rendering:** Most AI image generators render text poorly. Use "NO TEXT IN IMAGE" in every prompt and overlay text in Canva, Adobe Express, or Figma.
2. **Color consistency:** Every creative uses #faf4f1 (cream) as background. Never deviate to white (#ffffff) — the warmth is part of the brand.
3. **Orange accent:** #E8581A. Only on CTAs, arrows, accent lines, and key emphasis words. Never as background.
4. **Font sourcing:** Gilda Display (Google Fonts), Space Grotesk (Google Fonts), Inter (Google Fonts), Josefin Sans (Google Fonts) — all free.
5. **Batch production order:** Start with Creatives 13, 05, 03 (visual concepts) — these have the strongest scroll-stopping power and need the least copy to work. Test these first.
